How secrets are handled
  • Saved keys are never sent back to the browser — fields show a placeholder instead.
  • Leave a field blank to keep the stored value; use the button to erase it.
  • settings.json is written with owner-only permissions (0600).
  • Connection-test errors have key material redacted before display.
  • To rotate a key: paste the new value and save — the old one is overwritten immediately.
